# Unti Black Garlic Website

Production-ready static website for **Unti Black Garlic**, a premium black garlic brand from Surabaya, Indonesia.

## Pages

- `index.html` - Home / Beranda
- `about.html` - About Us / Tentang Kami
- `benefits.html` - Health Benefits / Manfaat Kesehatan
- `shop.html` - Shop & Contact / Toko & Kontak

## Features

- English and Bahasa Indonesia language toggle
- Responsive mobile, tablet, and desktop layouts
- SEO metadata for target keywords
- Local SVG visual assets for fast loading
- WhatsApp ordering CTA for `+62-813-9238-5171`
- Instagram and Shopee links
- Static hosting friendly: no build step required

## Run Locally

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Deploy

Upload the project folder to any static host such as Netlify, Vercel, GitHub Pages, Cloudflare Pages, or shared hosting. The entry page is `index.html`.

## Customization

Update the Instagram/Shopee URLs in the HTML files if the brand uses a different official storefront or social handle.
